# EzoOffice - Hızlı Deployment Script

Bu script, Google Cloud VM'lerde hızlı kurulum için hazırlanmıştır.

## Kullanım

### PostgreSQL VM Kurulum Scripti

`setup-postgres.sh` dosyasını PostgreSQL VM'ine yükleyin ve çalıştırın:

```bash
chmod +x setup-postgres.sh
sudo ./setup-postgres.sh
```

### Uygulama VM Kurulum Scripti

`setup-app.sh` dosyasını Uygulama VM'ine yükleyin ve çalıştırın:

```bash
chmod +x setup-app.sh
sudo ./setup-app.sh
```

**Not:** Script'leri çalıştırmadan önce içindeki IP adreslerini ve şifreleri kendi ortamınıza göre düzenleyin!
